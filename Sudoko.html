Richochet Robots Sudoko Solver

Sudoko Solver

A graphical interface game of Sudoko. I have built an solver for the puzzle, which works on "BACK Tracking algorithm". It was a fun side project.
I used the backtracking algorithm in this solver because if we try to find each and every solution and try to see if it works then it will take forever.
So, by doing some research and talking to some of my seniors, i used the back tracking algorithum. This Project is an product of the
bigger project like Richochet Robots. The management skills that i learned from the previous project encouraged me to built this one. This project is also hosted on GITHUB.
